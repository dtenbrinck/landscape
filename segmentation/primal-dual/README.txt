This git repository uses git submodules to build upon the 'thresholding'
repository and its code. These submodules have to be initialized and updated
manually. Hence use the following commands to use submodules correclty:

After cloning the superrepository initialize subrepositories with:
  -> git submodule init
  -> git submodule update


For updating subrepositories to the commit used in the superrepository use:
  -> git submodule update
